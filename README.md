RAG Chat по PDF-конспектам

Небольшое учебное MVP-приложение на Streamlit для поиска информации и получения ответов по PDF-конспектам с использованием подхода RAG (Retrieval-Augmented Generation).

Принцип работы: документ разбивается на фрагменты, для них создаются эмбеддинги, данные сохраняются в ChromaDB, по запросу находятся наиболее релевантные фрагменты, на основе которых формируется ответ с указанием страниц.

Что умеет:

Загружать один или несколько PDF-файлов
Парсить PDF постранично с сохранением метаданных source (файл) и page (страница)
Разбивать текст на чанки примерно по 1000 символов с overlap около 200
Строить и обновлять векторный индекс в ChromaDB (persist в ./chroma_store)
Находить релевантные фрагменты (top-k) и показывать их вместе со страницами
Генерировать ответ по найденному контексту при наличии OpenAI
Работать в режиме только поиска, если OPENAI_API_KEY не задан
